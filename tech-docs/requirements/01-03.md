# 01-03 Save Session

## Overview

Save all changes made to the current session.

## Functional Requirements

| ID | Requirement |
|----|-------------|
| FR-01-03-01 | The system shall save the current session to its existing location. |
| FR-01-03-02 | The system shall report an error if the session cannot be saved. |
| FR-01-03-03 | The system shall update the session's modified state after a successful save. |

## Preconditions

- A session is open.
- The session has a valid save location.

## Postconditions

- All changes are written to disk.
- The session is no longer marked as modified.
