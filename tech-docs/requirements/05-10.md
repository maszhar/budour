# 05-10 Set Editing Scope

## Overview

Select whether editing operations apply to the active region or all regions in the Piano Roll.

## Functional Requirements

| ID | Requirement |
|----|-------------|
| FR-05-10-01 | The system shall allow the user to select an editing scope. |
| FR-05-10-02 | The system shall provide the **Active Region** editing scope. |
| FR-05-10-03 | The system shall provide the **All Regions** editing scope. |
| FR-05-10-04 | The system shall apply editing operations according to the selected editing scope. |
| FR-05-10-05 | The system shall restore the previously selected editing scope when the Piano Roll editor is reopened. |
| FR-05-10-06 | The system shall default the editing scope to Active Region if no previous editing scope has been stored. |

## Preconditions

- A Piano Roll editor is open.

## Postconditions

- The selected editing scope is updated.
- Subsequent editing operations use the selected editing scope.
